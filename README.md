### Репозиторий содержит практические работы по дисциплине **«Основы программирования»**

### Язык: C

## Изученные темы

### Практическая работа №1 — Структура программы, основные типы данных, ввод/вывод
- Структура простой программы на C
- Основные типы данных (`int`, `double`, `char`, `unsigned char`, `short int`, `long int`, `unsigned int`, `float`)
- Ввод и вывод данных (`scanf()`, `printf()`)
- Форматирование вывода и спецификаторы формата
- Переполнение целочисленных типов и поведение `unsigned`
- Ошибки при работе с `scanf()` / `printf()`
- Порядок выполнения операций присваивания
- Расширенные операции присваивания (`+=`, `*=`) и инкремент/декремент

**Отчёт** → [`practical-1.pdf`](docs/practical-1.pdf)

### Практическая работа №2 — Ветвления и циклы
- Условные операторы (`if`, тернарный оператор)
- Циклы (`for`, `while`, `do-while`)
- Работа с математическими функциями (`math.h`)
- Проверка области определения функции
- Рекуррентные соотношения для рядов

### Практическая работа №3 — Указатели
- Основы работы с указателями
- Арифметика указателей
- Приведение типов указателей
- Анализ расположения переменных в памяти
- Порядок байт (Little-Endian)

### Практическая работа №4 — Массивы и динамическая память
- Одномерные и многомерные массивы
- Динамическое выделение памяти (`malloc`, `calloc`, `free`)
- Обработка матриц
- Поиск минимумов, фильтрация по условиям

### Практическая работа №5 — Функции
- Передача функции как параметра (callback)
- Численное интегрирование методом трапеций
- Работа с матрицами через функции
- Выделение и освобождение памяти в функциях

### Практическая работа №6 — Строки
- Работа с символьными массивами
- Функции `strtok`, `strchr`, `strcat`
- Обработка текста (фильтрация слов, форматирование)

### Практическая работа №7 — Файлы
- Работа с текстовыми файлами (`fopen`, `fscanf`, `fputs`)
- Построчная обработка текста
- Фильтрация строк по условиям
